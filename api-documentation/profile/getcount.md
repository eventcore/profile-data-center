# Get Profile Count

Get Profile Count.

## Overview

**URL** : `/api/Profile/$count`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : User can read Profiles  -- OR -- User has a SuperUser role

## Success Responses

**Condition** : User is authorized

**Code** : `200 OK`

**Content example** :

```json
12345
```
